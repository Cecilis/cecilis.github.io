Sobre Mí

Soy desarrolladora de software con más de una década de experiencia construyendo, manteniendo y evolucionando aplicaciones para distintos sectores. He trabajado de forma remota para empresas en Latinoamérica, integrando APIs REST/SOAP, optimizando bases de datos y automatizando despliegues.

A lo largo de mi trayectoria he utilizado C#, VB.NET, Python y JavaScript en diversos frameworks como .NET Core, ASP.NET MVC, WPF, Django, y también he trabajado con frontend en Angular, React y jQuery. En el backend he implementado microservicios, message brokers (RabbitMQ, MassTransit) y caching con Redis.

Me interesa la aplicación de inteligencia artificial para optimizar el desarrollo (Copilot, Gemini, DeepSeek, ChatGPT) y disfruto enfrentándome a desafíos técnicos que requieren aprendizaje continuo.
